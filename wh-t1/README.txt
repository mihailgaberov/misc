In order to be able to compile the project:
1. Install NodeJS - nodejs.org
2. Install Grunt cli - run in Command Prompt in the project folder "npm install -g grunt-cli"
3. In Command Prompt - "npm install" to install the necessary node modules.
4. Install Ruby to be able to compile SASS to CSS - https://www.ruby-lang.org/en/documentation/installation/
5. Install SASS - run "gem install sass" in Command Prompt in the project folder
6. Once the above is done:
    - "grunt watch" in cmd to watch the changes in html/css files
    - "grunt buildcss" to compile the css from sass
