In order to be able to compile the project:
1. Install NodeJS - nodejs.org
2. Install Grunt cli - run in Command Prompt in the project folder "npm install -g grunt-cli"
3. In Command Prompt - "npm install" to install the necessary node modules.
4. Once the above is done:
    - "grunt watch" in cmd to watch the changes in html/js files
    - "grunt" to minify the javascript files
